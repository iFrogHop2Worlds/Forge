Experimental database engine written in Rust using a LSM tree and SSTables for high write throughput.

Depending on the size of the data current test on my machine hit between 1 - 1.67 million writes per second. With the current implementation.

instruction to use the cli interface here https://github.com/iFrogHop2Worlds/Forge/blob/main/ForgeCli/readme.md

ForgeEngine docs can be generated by running `cargo doc --open` in the Forge/ForgeEngine directory. Will publish to docs.rs soon.
